# LEÇON VINGT-CINQ

# 形容词比较级

两个人 / 事物进行比较，有高于、相等、低于三种情况，分别在形容词前加 plus、aussi、moins

  • A + plus/aussi/moins + adj. + que（相当于 than） + B
  • A 高于 / 相等 / 低于 B
    • Paul est plus intelligent que Marie. 保尔比玛丽聪明
  • 说明：

    • 形容词的性、数要与比较的第一成分一致

    • 比较的第二成分可以是名词、代词或其他句子成分

      • En hiver, il fait moins froid à Shanghai qu'à Beijing. 冬天，上海的天气没有北京这么冷

    • 比较的第二成分有时可以不表达出来

      • Ce vin est plus cher. 这种酒更贵一些

    • 形容词 bon 比较级特殊：

      • Mon stylo est meilleur que votre stylo. 我的钢笔比您的钢笔好
原形比较级
bonmeilleur
bonnemeilleure

# 副词比较级

  1. 与形容词相同，分别加 plus、aussi、moins
    • Vous courez plus vite que moi. 您比我跑得快
  2. 副词 bien、beaucoup、peu 的比较级有特殊形式
    • Je parle peu en classe, mais il parle encore moins que moi. 我在课堂上发言不多，但他发言比我还少
原形较高相等较低
bienmieuxaussi bienmoins bien
beaucoupplus (s 发音)autant
peumoinsaussi peu

# 形容词最高级

le(la,les) plus + adj.表示最高程度
le(la,les) moins + adj.表示最低程度

# LEÇON VINGT-SEPT

# 关系从句和关系代词 qui

关系代词 qui 在关系从句中一般作主语，既可指人，也可指物。其所代替的词称为先行词，关系代词一般紧跟在先行词后面。关系从句中动词要与先行词性、数一致。

  • Faites ces exercices, qui ne sont pas difficiles. 请做这些练习，它们不难
  • L'épicier qui est tout près de chez vous peut vous aider. 离您家很近的食品杂货商可以帮助您

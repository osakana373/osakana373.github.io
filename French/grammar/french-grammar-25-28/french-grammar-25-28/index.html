# LEÇON VINGT-CINQ

# 形容词比较级

两个人 / 事物进行比较，有高于、相等、低于三种情况，分别在形容词前加 plus、aussi、moins

  • A + plus/aussi/moins + adj. + que（相当于 than） + B
  • A 高于 / 相等 / 低于 B
    • Paul est plus intelligent que Marie. 保尔比玛丽聪明
  • 说明：

    • 形容词的性、数要与比较的第一成分一致

    • 比较的第二成分可以是名词、代词或其他句子成分

      • En hiver, il fait moins froid à Shanghai qu'à Beijing. 冬天，上海的天气没有北京这么冷

    • 比较的第二成分有时可以不表达出来

      • Ce vin est plus cher. 这种酒更贵一些

    • 形容词 bon 比较级特殊：

      • Mon stylo est meilleur que votre stylo. 我的钢笔比您的钢笔好
原形比较级
bonmeilleur
bonnemeilleure

# 副词比较级

  1. 与形容词相同，分别加 plus、aussi、moins
    • Vous courez plus vite que moi. 您比我跑得快
  2. 副词 bien、beaucoup、peu 的比较级有特殊形式
    • Je parle peu en classe, mais il parle encore moins que moi. 我在课堂上发言不多，但他发言比我还少
原形较高相等较低
bienmieuxaussi bienmoins bien
beaucoupplus (s 发音)autant
peumoinsaussi peu

# 形容词最高级

le(la,les) plus + adj.表示最高程度
le(la,les) moins + adj.表示最低程度
  • Paul n'est pas grand. C'est le moins grand des élèves. 保尔个子不高，他是这些学生中个子最矮的

形容词 bon 的最高级形式：

  • C'est le meilleur film de l'année. 这是今年最优秀的一部电影
阳性原形最高级阴性原形最高级
bonle meilleurbonnela meilleur
bonsles meilleursbonnesles meilleures

说明：

  1. 形容词最高级后面一般要加补语 de，“... 之中最...“
    • C'est le plus âgé de ces ouvriers. 他是这些工人中最年长的一位
  2. 最高级形容词放在名词前，名词不必重复使用定冠词；放名词后面时，则要重复定冠词
    • C'est le plus long fleuve
    • La Loire est le fleuve le plus long de la France.
  3. 表示 “最... 之一 “用” un (une) des plus/moins“
    • Beijing est une des plus grandes villes du monde. 北京是世界最大的城市之一
  4. 定冠词有时用主有形容词代替，并省略补语
    • C'est mon meilleur ami. 这是我最要好的朋友

# 副词最高级

  1. 与形容词最高级略有不同
    • C'est Pascal qui court le plus vite. Pascal 跑得最快
le (不变) plus + adv.表示最高程度
le (不变) moins + adv.表示最低程度
  1. beaucoup、peu、bien 等副词的最高级形式特殊：
    • Qui chante le mieux dans votre classe? 你们班谁唱歌唱的最好？
原形最高级释义
beaucouple plus最多
peule moins最少
bienle mieux最好

# LEÇON VINGT-SIX

# 被动态

être + 过去分词 + par/de + 施动者补语

  1. 被动态由助动词 être + 过去分词构成，过去分词要与主语性、数一致
    • La voiture est réparée par un de mes amis. 汽车是我的一位朋友修理的
  2. 在被动态中，动作的实施者称为施动者补语，它通常由介词 par/de 引出。施动者补语有时可以省略
    • Le repas est préparé par Marie. 饭菜是玛丽准备的
    • Cet écrivain est aimé de tous les lecteurs. 这位作家得到所有读者的喜爱

注意：par 一般强调动作，de 则强调状态或用在表示感情、思想活动的动词后面

# 形容词前 de 代替 des

不定冠词 des 后紧跟形容词时，des 一般改为 de
+ Nous avons des fleurs.
+ Nous avons de belles fleurs.

# LEÇON VINGT-SEPT

# 关系从句和关系代词 qui

关系代词 qui 在关系从句中一般作主语，既可指人，也可指物。其所代替的词称为先行词，关系代词一般紧跟在先行词后面。关系从句中动词要与先行词性、数一致。

  • Faites ces exercices, qui ne sont pas difficiles. 请做这些练习，它们不难
  • L'épicier qui est tout près de chez vous peut vous aider. 离您家很近的食品杂货商可以帮助您

# 关系代词 que

  1. que 用以引导关系从句，通常作从句动词的直接宾语
    • Quels sont les monuments que vous aimez à Paris? 您喜欢巴黎的哪些建筑？（que = les monuments，是 aimez 的直接宾语）
  2. que 本身无词形变化，但 que 引导的从句中，若从句动词是以 avoir 助动词的复合时态，过去分词与先行词的性、数一致
    • Voilà la moto que j'ai achetée. 这就是我买的摩托车
  3. 说明：
    1. 同关系代词 qui 一样，que 引导的从句也分为限定性从句和非限定性从句，非限定性从句一般用逗号隔开
    2. 法语中使用哪个关系代词，取决于先行词在从句中所起的语法作用（主语、宾语），而英语则往往取决于先行词的性质（人、物）
    3. 关系代词 que 相当于英语中的 whom、which 或 that
      • C'est un homme que tout le monde respecte.
      • He is a man whom everybody respects.
    4. 法语的关系代词不可省略

# 关系代词 où

où 在引导的关系从句中作地点状语或时间状语，它的先行词只能是物
+ La ville où je suis né est un port. (où = dans cette ville，作从句的地点状语) 我出生的那座城市是一个港口

说明： où 的用法相当于英语中的 when 或 where，不可省略

# 关系代词 dont

dont 代替一个带有介词 de 的名词，这个名词作它的先行词，可以是人或物

  1. 动词的间接宾语
    • Voilà le docteur dont je vous ai parlé. (dont = de ce docteur) 这就是我和您讲过的那位医生
  2. 名词补语
    • J'ai vu un film dont l'histoire est très intéressante.(dont = de ce film) 我看了一部故事情节很有趣的电影
  3. 形容词补语
    • Il nous a parlé de son travail dont il est très satisfait. (dont = de son travail)
    • 常用的形容词还有：
      • (être) content de: 为... 高兴
      • (être) fier de: 为... 自豪

注意：dont 的用法与英语中的 whose、of whom、of which 相似

# LEÇON VINGT-HUIT

# 指示代词

指示代词一般用于代替上文提到的人或事物，以避免名词的重复。指示代词分为简单词形和复合词形，有性、数变化

# 简单词形

阳性阴性中性
单数celuicellece
复数ceuxcellesce

简单词形的指示代词后要跟随一个限定性的补语，一般是以介词 de 引导的补语或关系从句

  1. de + 名词补语
    • Voilà mon vélo, où est celui de Pierre? (celui = le vélo) 这是我的自行车，Pierre 的在哪？
  2. 指示代词 + 关系从句
    • Ces journaux ne sont pas ceux qui sont sur la table ce matin. (ceux = les journaux) 这些报纸不是今早桌子上的那些报纸

说明：celui qui...，ceux qui... 可以不代替前面的名词，而用于指某一类人（不能指物）
+ Ceux qui ne travaillent pas ne reçoivent rien. 不劳动者不收获

# 复合词形

阳性阴性中性
单数celui-cicelle-ciceci
celui-làcelle-là
复数ceux-cicelles-cicela
ceux-làcelles-là
  1. 复合指示代词在上下文清楚的情况下单独使用，不需要加限定性的补语
    • J'ai deux ordinateurs, mais celui-ci ne marche pas très bien. 我有两台电脑，但这一台不太好用
  2. 在指明 “这个、那个，这些、那些” 时，较近者 / 指后出现的人或物用 celui-ci、ceux-ci，较远者 / 指先出现的人或物用 celui-là、ceux-là
    • Ces deux maisons sont très belles, mais celle-ci est petite, celle-là est grande. 这两座房子很漂亮，但这一座小，那一座大

# 中性指示代词 ce、ceci、cela

  1. ce 的用法
    1. ce 作主语是，动词 être 可以用单数，也可以用复数
    2. ce 不作主语时，后面一般要跟关系从句
      • Je ne comprends pas ce que vous avez dit. 我不明白您说的是什么意思
  2. ceci 和 cela (ça) 的用法
    1. 通常在口语中使用，用于代替交谈双方都清楚的事物
      • Ceci est à moi, cela est à toi. 这是我的，那是你的
    2. 可单独使用，用来指物
    3. cela 代替一个词组或一个句子
      • Il est malade? Qui t'a dit cela(ça)?
  3. cela 日常生活中常缩合成 ça 的形式，用于习惯用语

# 简单将来时

# 构成

  1. 第一组和第二组动词在原形动词后直接加下列词尾：
    • -ai,-as,-a,-ons,-ez,-ont
parler
je parlerainous parlerons
tu parlerasvous parlerez
il parlerails parleront
finir
je finirainous finirons
tu finirasvous finirez
il finirails finiront
  1. 以 - re 结尾的第三组动词变成简单将来时，要先去掉原形动词词尾的 e，然后加上述词尾
lire
je lirainous lirons
tu lirasvous lirez
il lirails liront
  1. 但大多数第三组动词变成将来时，词根有变化，词尾仍使用 - ai,-as,-a,-ons,-ez,-ont
原形简单将来时原形简单将来时
avoirj'aurairecevoirje recevrai
êtreje seraisavoirje saurai
allerj'iraivenirje viendrai
faireje feraivoirje verrai
devoirje devraivouloirje voudrai
courirje courraifalloiril faudra
pouvoirje pourraipleuvoiril pleuvra

# 用法

用于表示近期 / 遥远的将来发生的动作，或表示在未来的时间内将要存在的状态，这个将来可能是比较近的将来，也可能是比较远的将来，但一般情况下不会在马上发生的事情时使用

  • Il fera beau demain. 明天天晴

简单将来时还会用来代替直陈式或命令式，来表达委婉的语气

  • Vous fermerez la porte avant de partir. 走之前请您关门

# 先将来时

# 构成

先将来时是一种复合时态，由助动词 avoir 或 être 的简单将来时加动词的过去分词构成

finir
j'aurai fininous aurons fini
tu auras finivous aurez fini
il aura finiils auront fini
partir
je serai parti(e)nous serons partis(es)
tu seras parti(e)vous serez parti(e)(s)(es)
il sera partiils seront partis
elle sera partieelles seront parties

# 用法

先将来时是一个完成时态，用于表示在另一个将来的动作发生之前已经完成的动作
+ Dès que nous serons arrivés à Paris, nous vous écrirons. 我们一到巴黎就给你们写信

说明：

  1. 在复合句中使用简单将来时和先将来时，目的是强调两个动作之间的先后关系，若不强调这种关系，两个动作均可使用简单将来时
    • Ce soir, je lirai les journaux et je regarderai la télévision. 今晚我要看报纸和看电视
  2. 先将来时有时也可在独立句中使用，表示即将迅速完成的动作
    • J'aurai fini mon article dans deux minutes. 我一会儿就能把文章写完
